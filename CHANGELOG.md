# Changelog

## 1.0.0 - 2026-06-29

- Initial production-grade project scaffold.
- Added modular installer entrypoint and interactive menu.
- Added shared libraries for logging, OS detection, package installation, backups, downloads, verification, and UI helpers.
- Added modules for productivity tools, modern CLI tools, shell, tmux, window manager, Python, Go, Docker, fonts, red team tools, update, repair, verify, backup, restore, and uninstall.
- Added configuration files for zsh, tmux, i3, polybar, picom, rofi, dunst, kitty, and alacritty.
- Added tests and GitHub Actions CI workflow.
- Added architecture and operations documentation.
